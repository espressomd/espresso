Explanation of the "scripts/main.tcl" script

The mbtools packages

  • analysis : A framework for organizing analysis of simulation results. Includes many analysis routines specifically designed for lipid membranes.
  • system_generation : A package for setting up lipid membrane systems in a variety of geometrical shapes.
  • utils : Useful utilities routines of various types. Includes file management, basic geometry and math proceedures and many routines for dealing with topologies.

Other packages

  • mmsg : A standard output wrapper used by mbtools for controlling warning,error,debug and standard messages.
